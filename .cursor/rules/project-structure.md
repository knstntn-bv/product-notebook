# Структура проекта и соглашения

## Структура файлов

- Компоненты: `src/components/`
- Страницы: `src/pages/`
- Хуки: `src/hooks/`
- Утилиты: `src/lib/`
- Контексты: `src/contexts/`
- Интеграции: `src/integrations/`

## Коммиты и изменения

- При изменении БД всегда выполняй все 3 шага для миграций (см. `migrations.md`)
- При добавлении новых фич обновляй документацию в `docs/` если нужно
- Следуй существующим соглашениям по именованию
