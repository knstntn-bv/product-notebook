# Проблема

Сейчас для создания фичи из гипотезы используется кнопка, размещённая непосредственно в столбце таблицы. У этого решения есть 2 недостатка:
- это единственный Action в таблице, неконсистентно;
- слишком накладно с точки зрения занимаемого пространства в и так громоздкой таблице.

# Решение

1. Убрать кнопку и столбец Actions из таблицы гипотез.
2. Разместить кнопку в правой панели редактора гипотез, в блоке кнопок. Поведение (в рамках этого требования) останется прежним.
