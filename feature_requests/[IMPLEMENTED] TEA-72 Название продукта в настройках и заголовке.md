# Проблема

В приложении используется сущность `product` с полем `name`, но пользователь не может изменить название продукта из интерфейса.  
Кроме того, в шапке показывается статичный текст `Product Notebook`, из-за чего не видно, с каким продуктом сейчас ведётся работа.

# Решение

Добавить управление названием текущего продукта в UI и вывести это название в шапке приложения:

1. В `Project Settings` добавить поле редактирования названия текущего продукта (`products.name`) с валидацией:
   - значение обязательно после `trim`;
   - ограничение длины: 1..100 символов;
   - сохранение доступно только при валидном и изменённом значении.
2. Добавить пункт открытия настроек проекта в меню `Settings`.
3. В заголовке страницы вместо статичной строки `Product Notebook` показывать название текущего продукта.
4. Если название недоступно (загрузка/пустое состояние), использовать фолбэк `Product Notebook`.
5. Обновить документацию по настройкам и шапке приложения.
