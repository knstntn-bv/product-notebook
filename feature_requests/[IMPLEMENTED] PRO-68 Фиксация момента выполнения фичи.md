Техническое требование, понадобится для реализации будущей функциональности.

Необходимо добавить к сущности фичи поле даты-времени её закрытия. Поле необязательное, по умолчанию пустое.

Моментом закрытия считается перенос в колонку Done или Cancelled - при этом событии следует установить текущее время в поле времени закрытия.

Поле после первой записи не фиксируется: в случае повторного обнаружения события переноса в одну из колонок Done или Cancelled поле обновляется на текущую дату.
